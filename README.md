# Calculator Project

A simple Python calculator application for learning Git and GitHub collaboration.

## Features
- Basic arithmetic operations (add, subtract, multiply, divide)
- Power calculation
- Error handling for division by zero
- Unit tests included

## Usage
```bash
python calculator.py
```

## Testing
```bash
python test_calculator.py
```
